Данная программа осуществляет упаковку файла с последовательностями
РНК и ДНК в бинарный формат из .fasta формата и обратно.

Программа требует следующих параметров:
ключ -i указываем входящий файл,
ключ -o указываем файл выхода,
ключ -d задаёт направление преобразования файла:
         pack   - упаковывать в бинарный формат,
         unpack - распаковывать обратно.

По завершении программа всегда возвращает 0 вне зависимости от того,
удачно или не удачно прошла упаковка/распаковка
